To install Project Starfigther download "Setup Files.rar" from:

http://csce.uark.edu/~bgooding/Team15/

or

from the downloads section on github.

Install note: You must be using a windows based PC.

Install Instructions:
1) Extract the contents of Setup Files.rar to a temporary folder.
2) Run setup.exe
3) If you do not have the required packages they will be downloaded and install.
4) A shortcut to project starfigther will be on your desktop and in the start menu

Uninstall instruction:
1) Open my computer
2) click the option to uninstall/change programs
3) select Project Starfighter from the list.
